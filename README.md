# 🏪 DZ Vent & Stock

نظام مبيعات ومخزون متكامل للمحلات التجارية الجزائرية.

## 🚀 تشغيل المشروع

### المتطلبات
- Node.js v18+
- npm

### الخطوات

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل في وضع التطوير
npm start

# 3. بناء ملف EXE
npm run build
```

## 📁 هيكل المشروع

```
DZVentStock/
├── main.js          ← Electron main process
├── preload.js       ← IPC bridge
├── package.json
├── db/
│   └── database.js  ← SQLite (better-sqlite3)
└── renderer/
    └── index.html   ← الواجهة الكاملة
```

## ⌨️ اختصارات لوحة المفاتيح

| المفتاح | الوظيفة |
|---------|---------|
| F1 | لوحة التحكم |
| F2 | نقطة البيع (POS) |
| F3 | المنتجات |
| ESC | إغلاق النافذة المنبثقة |

## 🗄️ قاعدة البيانات

SQLite محلي — الملف في:
`%APPDATA%\dz-vent-stock\dzventstock.db`
